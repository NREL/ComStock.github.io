# Published Datasets
{: .fw-500 }
These datasets describe the timeseries energy consumption of the U.S. commercial building stock at the end-use level. For details on how it was created and validated, please see the project’s [final report](https://www.nrel.gov/buildings/end-use-load-profiles.html). Visit the [Accessing Data]({{site.baseurl}}{% link docs/data/accessing_data.md %}) page for more details about the data.
